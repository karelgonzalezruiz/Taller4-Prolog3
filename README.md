# Taller-Prolog-Misiones
Resolución del taller 4.

Que hice en este taller:

* **Base de Conocimiento:** Definición de personajes, misiones, inventarios y objetos requeridos para completar ciertas misiones.
* **Validación de Misiones:** Regla para verificar si un personaje puede aceptar una misión según su nivel y la dificultad requerida.
* **Recursión y Experiencia:** Cálculo recursivo de la XP acumulada según el nivel del personaje.
* **Inventarios y Listas:** Verificación de objetos en inventarios y fusión de inventarios para trabajar con equipos.
* **Reportes Conjugados:** Uso de tiempos, personas y números para conjugar el verbo “ser” y generar mensajes automáticos.
* **Misiones Individuales o Grupales:** Implementación de una regla para iniciar misiones con una persona o un grupo, validando nivel, requisitos y XP final de cada participante.
